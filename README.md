# **Análise de Dados de Educação – Rede de Ensino de Pernambuco**

Este projeto foi desenvolvido para a disciplina **Projetos 5**, da **Cesar School**.

A análise exploratória dos dados foi realizada em duas etapas:
- **Python (Google Colab):** execução do tratamento inicial, exploração estatística e preparação dos dados.  
- **Power BI:** desenvolvimento da análise visual e criação dos dashboards para interpretação dos resultados.
